# IronManTeamTools

本机环境 Python 3.10.3

## 终端使用者
1. pip install -r requirements.txt
2. 安装nodejs
3. 解压zip文件 python crypto_00_check_gui.py

## 开发者环境部署
1. 安装好nodejs环境。
2. pip install -r requirements.txt
3. 命令行执行 npm install -g pnpm
4. 进入 client 目录 执行 pnpm install

build.bat: 开发者build一键部署加启动
